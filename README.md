# GoodWe-Tibber-Smart-Charging-Automatisiertes-Laden-Eigenverbrauch
Steuert deinen GoodWe Batteriespeicher intelligent basierend auf dynamischen Tibber-Strompreisen.     Der Akku wird automatisch in den günstigsten Stunden (unter Berücksichtigung eines Preis-Limits) geladen und dient sonst dem Eigenverbrauch (kein Export ins Netz).
